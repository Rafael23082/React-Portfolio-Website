This is a simple portfolio website built using React and Tailwind CSS. <br />
The website consists of:
<ul>
  <li>Landing page</li>
  <li>About page</li>
  <li>Projects page</li>
  <li>Contact page</li>
</ul>
How to run:
<ul>
  <li>Clone the repository</li>
  <li>Run npm install</li>
  <li>Run npm run dev</li>
</ul>
